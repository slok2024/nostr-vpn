# nvpn

CLI and daemon for Nostr VPN private mesh networks.

The `nvpn` binary can initialize Nostr VPN configuration, run the VPN session
foreground or as a daemon, install/manage the system service on supported
platforms, and perform update checks for release binaries.

Install from crates.io:

```bash
cargo install nvpn
```

Run `nvpn --help` for the current command set.
